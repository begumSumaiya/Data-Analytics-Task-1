# Data Cleaning and Preprocessing - Task 1

## Objective

Clean and preprocess a raw Coffee Sales dataset by identifying and resolving common data quality issues using Microsoft Excel.

## Dataset

- **Dataset:** Coffee Sales (`index_1.csv`)
- **Records Used:** First 2,000 rows

## Tools

- Microsoft Excel
- GitHub

## Cleaning Performed

- Checked for missing values
- Checked for duplicate records
- Removed unnecessary spaces from text fields
- Standardized column names
- Verified data types
- Reviewed date formats where applicable

## Files

- `index_1.csv` – Original dataset
- `index_1_cleaned_2000.csv` – Cleaned dataset
- `Data_Cleaning_Summary.md` – Summary of the cleaning process

## Result

The dataset was cleaned, standardized, and prepared for future data analysis.
