# Data Cleaning Summary

## Dataset

**Dataset:** Coffee Sales Dataset (`index_1.csv`)

## Objective

The objective of this task was to clean and preprocess a raw dataset to improve its quality and prepare it for future data analysis.

## Scope

The original dataset contained a large number of records. For this task, the first **2,000 rows** were selected to demonstrate the complete data cleaning workflow while keeping the dataset manageable for analysis in Microsoft Excel.

## Data Cleaning Process

The following steps were performed:

- Reviewed the dataset for missing (null) values.
- Checked for duplicate records.
- Removed leading and trailing spaces from text fields where applicable.
- Standardized column headers using lowercase letters and underscores.
- Reviewed date columns and ensured a consistent format where applicable.
- Verified that each column contained the appropriate data type.
- Saved the cleaned dataset as a separate file to preserve the original dataset.

## Findings

- No duplicate records were identified within the selected 2,000 rows.
- No significant missing values were found.
- Column names were standardized for consistency.
- Text values were cleaned by removing unnecessary spaces.
- The dataset was reviewed and prepared for future analysis.

## Output Files

- Original dataset: `index_1.csv`
- Cleaned dataset: `index_1_cleaned_2000.csv`

## Conclusion

The selected sample of the Coffee Sales dataset was successfully cleaned and standardized. The resulting dataset is organized, consistent, and ready for further analysis and visualization.
